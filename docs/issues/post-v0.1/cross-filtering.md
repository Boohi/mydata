## Goal

Clicking any element in any of the four views (Today, Apps, World Map, Timeline) filters all other views simultaneously to that selection context.

## Why

Cross-view filtering turns mydata from four separate charts into a coherent investigation tool. Selecting a country on the map should immediately show which apps drove traffic there in the Today and Apps views.

## Scope

In:
- Global filter state (app, country, hostname, time range) shared across all views.
- Each view both reads and writes the filter state.
- Filter pills displayed at top of dashboard with clear-all action.
