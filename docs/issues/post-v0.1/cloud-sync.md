## Goal

Allow users to sync their network history across multiple Macs via end-to-end encrypted cloud storage as an optional paid feature.

## Why

Power users with multiple machines want unified history. E2E encryption ensures the cloud provider never sees plaintext data, preserving the privacy promise.

## Scope

In:
- Client-side encryption before any data leaves the device.
- Sync protocol that merges events from multiple sources without a central plaintext store.
- Paid tier gate and subscription management.
