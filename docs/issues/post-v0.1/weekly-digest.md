## Goal

Generate a formatted weekly summary of network activity as a PDF or HTML report, produced locally with no network access.

## Why

Privacy-conscious users want a regular human-readable record they can archive, share with a tech-savvy friend, or reference during an incident review.

## Scope

In:
- Scheduled weekly generation (or on-demand via Settings).
- Summary includes: top apps by bytes, top countries, notable new destinations, flow count.
- Output saved to `~/Documents/mydata-digests/` as HTML (PDF via print-to-PDF if desired).
